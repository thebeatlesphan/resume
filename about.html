2D Predator Prey Simulation
Java

Pure Java program that displays an ASCII simulation in the terminal. Focused on learning important object oriented programming and data structure concepts (polymorphism, inheritance, encapsulation, and arrays). Predator and prey are initially generated on a 200x200 grid and the user presses a key to move forward in time. Each organism performs one action (move, devour, or breed) a turn.

Natural Language Processor
Java

Pure Java program that implements a first-order Markov chain to determine which language a document is. Focused on learning implementation of Java classes, 2-D arrays, simple text processing, and basic numerical computing. The user feeds the program corpus files to study letter frequencies for a specific language and utilizes that to calculate the likelihood of which language the new string / file is.
